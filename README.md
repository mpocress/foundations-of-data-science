# Foundations of Data Science

A 6-part series of hands-on Jupyter notebooks that progressively teaches data science from Python fundamentals through advanced machine learning models.

## Notebooks

| # | Notebook | Topics |
|---|----------|--------|
| 0 | [Table of Contents](00_table_of_contents.ipynb) | Series overview, learning path |
| 1 | [Python and Statistics Foundations](01_python_and_stats_foundations.ipynb) | Python essentials, NumPy, Pandas, descriptive statistics, visualization |
| 2 | [Web Scraping and EDA](02_web_scraping_and_eda.ipynb) | HTTP requests, HTML parsing, data cleaning, exploratory data analysis |
| 3 | [Regression Fundamentals](03_regression_fundamentals.ipynb) | kNN regression, simple and multiple linear regression, residual analysis |
| 4 | [Polynomial and Regularized Regression](04_polynomial_and_regularized_regression.ipynb) | Polynomial features, overfitting, LASSO, Ridge, cross-validation |
| 5 | [Classification Models](05_classification_models.ipynb) | Logistic regression, kNN classification, confusion matrices, ROC curves |
| 6 | [Ensemble Methods](06_ensemble_methods.ipynb) | Decision trees, bagging, random forests, AdaBoost, gradient boosting |

## Prerequisites

- Basic Python experience helpful but not required
- Python 3.9+
